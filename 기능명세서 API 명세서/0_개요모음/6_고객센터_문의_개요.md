# 6. 고객센터 · 문의 도메인

공지사항, FAQ/문의(QNA), 챗봇, 관리자 문의 답변.

## 관련 DB 테이블
- `notice` (title, content, is_pinned, deleted_at)
- `faq` (type ORDER/SHIPPING/PRODUCT/OTHER, title, content)
- `qna` (type ORDER/SHIPPING/PRODUCT/OTHER, title, content, attachment_url, is_answered, member_id)
- `qna_answer` (content, answered_at, qna_id UNIQUE) — 문의당 답변 1건

## 화면 (기능정의서)
- **공지사항게시판** (사용자) — 목록/상세, is_pinned 상단 고정
- **문의하기** (사용자) — FAQ + 내 문의 작성/조회/수정/삭제 (답변 달리면 수정·삭제 불가)
- **고객센터 챗봇페이지** (사용자) — 질문→AI 답변
- **관리자_QNA관리** — 문의 목록/상세/답변 등록(→ is_answered=true)

## API
| 메서드 | 경로 | 설명 |
| --- | --- | --- |
| GET | /cs/notice/list | 공지 목록 |
| GET | /cs/notice/{id} | 공지 상세 |
| GET | /cs/qna/faq | FAQ 목록 |
| GET | /cs/qna/list | 내 문의 목록 |
| POST | /cs/qna | 문의 등록 [신규] (type/title/content/attachment_url) |
| GET | /cs/qna/{id} | 문의 상세(+답변) |
| PATCH | /cs/qna/{id} | 문의 수정 |
| DELETE | /cs/qna/{id} | 문의 삭제 |
| POST | /chatbot/ask | 챗봇 질문 |
| GET | /admin/qna | (관리자) 문의 리스트 |
| GET | /admin/qna/{id} | (관리자) 문의 상세 |
| POST | /admin/qna/{id}/answer | (관리자) 답변 등록 |

> 미확정 갭: 공지(notice)는 사용자 조회 API만 있고 **관리자 공지 CRUD API가 아직 없음**.
