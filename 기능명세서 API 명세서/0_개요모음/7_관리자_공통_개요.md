# 7. 관리자 공통 도메인

특정 도메인에 속하지 않는 관리자 진입/요약 화면. (도메인별 관리자 기능은 각 도메인 폴더에 있음: 상품→2, 주문→4, 쿠폰→5, 회원→1, 문의→6)

## 화면 (기능정의서)
- **관리자_로그인** — 관리자 계정 로그인 (member.role = ADMIN)
- **관리자_대시보드** — 오늘 주문/매출, 신규 회원, 재고 부족, 답변 대기 문의 요약

## API
| 메서드 | 경로 | 설명 |
| --- | --- | --- |
| POST | /admin/login | 관리자 로그인 |
| GET | /admin/main | 대시보드 요약 지표 |

> 대시보드 지표는 저장 컬럼이 아니라 각 테이블 집계(파생)입니다.
