# 1. 회원 · 인증 도메인

SnackDeal(과자 쇼핑몰)의 **회원 가입/로그인/내 정보/관리자 회원관리** 영역. 이 폴더 하나로 회원 기능을 파악할 수 있도록 기능정의서와 API 명세를 함께 모았습니다.

## 관련 DB 테이블
- `member` (id, email UNIQUE, password, name, phone, birth DATE, gender MALE/FEMALE, status ACTIVE/INACTIVE/DELETED, role USER/ADMIN, last_login …)
- `email_verification` (email, code, verification_token, verified, code/token 만료시각)

## 사용자 화면 (기능정의서)
- **회원가입** — 이메일 인증 후 가입. 입력: 이메일/비밀번호/이름/휴대폰(phone)/생년월일(birth)/성별(gender)
- **이메일인증** — 인증코드 발송→검증→`verification_token` 발급, 가입 시 서버 재검증
- **로그인** — 이메일+비밀번호, `last_login` 갱신
- **회원정보 관리** — 비밀번호 확인 후 휴대폰/비밀번호 변경 (주소는 4번 주소록에서 별도)
- **마이페이지 메인** — 주문/쿠폰/주소록/회원정보 진입 허브

## API
| 메서드 | 경로 | 설명 |
| --- | --- | --- |
| POST | /member/join | 회원가입 (email_verification_token 필수) |
| POST | /member/email/send-code | 인증코드 발송 |
| POST | /member/email/verify-code | 인증코드 검증 → verification_token 발급 |
| POST | /member/login | 로그인 |
| POST | /member/logout | 로그아웃 |
| POST | /member/token/refresh | 액세스 토큰 재발급 |
| GET | /member/me | 내 정보 조회 |
| PATCH | /member/me | 내 정보 수정 |
| GET | /admin/members | (관리자) 회원 리스트 |
| GET | /admin/members/{id} | (관리자) 회원 상세 |
| PATCH | /admin/members/{id}/status | (관리자) 회원 상태 변경 ACTIVE/INACTIVE/DELETED |
