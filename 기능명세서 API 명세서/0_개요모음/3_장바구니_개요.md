# 3. 장바구니 도메인

로그인 사용자의 장바구니. **옵션 없음** → 상품+수량 단위(cart_item은 member+product 단위, 재담기 시 수량 합산).

## 관련 DB 테이블
- `cart_item` (id, quantity, member_id, product_id, created_at, updated_at)

## 화면 (기능정의서)
- **장바구니** — 담은 상품 목록/수량 변경/선택·전체 삭제, 재고 연동, "주문하기"로 결제 이동

## API
| 메서드 | 경로 | 설명 |
| --- | --- | --- |
| GET | /cart | 장바구니 조회 |
| POST | /cart | 장바구니 담기 (product_id, quantity) |
| PATCH | /cart/{item_id} | 수량 변경 |
| DELETE | /cart | 개별/선택 삭제 (item_ids, 없으면 전체) |
