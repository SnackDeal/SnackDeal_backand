# 상품리스트페이지

사용자/관리자 구분: 사용자
연관화면: 상품 페이지
도메인: user/product
기능설명: 상품 리스트를 조회한다
우선순위: 상
API : /product/list (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/product%20list%200a0bdfb849fa8249bbcd0113f6d35ee1.md)

### 선행조건

- 없음(비회원도 가능)

### 트리거

- 카테고리 진입 / 검색어 입력 / 페이지 접속

### 동작

- 메인 : 최신 상품 or 탑5정도 해서 하는걸 고려
- 등록된 상품 목록을 표시한다 (썸네일, 상품명, 가격, 품절 배지, 품절임박(임의로(10개미만), 시간남으면 관리자에서 설정) 배지)
- 재고 노출 안함
- 정렬 옵션 제공
    - 최신순 / 가격 낮은 순 / 가격 높은 순 / 인기순(주문 수 기반)
- 카테고리 필터로 목록을 필터링한다
- 검색어(상품명 기준)로 검색한다
- 페이지네이션 (페이지당 20~30개)
- 품절 상품은 별도 배지로 표시 (구매 자체는 상세 페이지에서 차단)

### 예외 / 제한사항

- 조회 결과가 없을 경우 빈 결과 안내 문구를 표시한다
- (선택)판매중지 상품은 목록에 노출하지 않는다
