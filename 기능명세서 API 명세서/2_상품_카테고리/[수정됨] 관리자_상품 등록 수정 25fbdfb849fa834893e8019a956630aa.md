# 관리자_상품 등록 / 수정 (수정됨)

사용자/관리자 구분: 관리자
연관화면: 상품관리
도메인: admin/product
기능설명: 기존 관리자가 상품관리 상세페이지에 진입한다
우선순위: 상
API : /admin/product (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20product%209c9bdfb849fa826599eb01a76b8f3d34.md), /admin/product/{id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20product%20%7Bid%7D%203c8bdfb849fa8384893e818b8519eddf.md), /admin/product/{id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20product%20%7Bid%7D%20b59bdfb849fa83a885e7012ce0ae379c.md), /admin/product/{id}/status (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20product%20%7Bid%7D%20status%20034bdfb849fa831b8afa8140e4153eb6.md)

### 선행조건

- 관리자 로그인 상태여야 한다

### 트리거

- "상품 등록" 버튼 클릭 / 리스트에서 상품명 클릭 (수정)

### 동작

[상품 상세 정보](https://app.notion.com/p/390bdfb849fa80228c48d37827fcd31e?pvs=21) (샘플. 이미지는 product_image 테이블에 상품당 1행 저장)

- 상품 기본 정보 입력
    - 상품명, 판매가, 카테고리(드롭다운 선택), 상품 상세 설명
    - 대표 이미지 1장 (필수)
- 재고 정보 입력 (본 화면 내 통합 관리)
    - 재고 수량: 0 이상의 정수 입력
    - 재고 0일 경우 사용자 상품 상세 페이지에서 자동으로 "품절" 처리 및 구매 차단
- 판매 상태 설정 (판매중 / 판매중지)
    - 판매중지 시 사용자 리스트/상세 페이지에 노출되지 않음
- 저장 시 유효성 검증 후 DB에 반영하고 상품리스트페이지에 즉시 반영한다
- 수정 시 기존 값을 폼에 채워서 보여준다

### 예외 / 제한사항

- 필수 입력값(상품명, 판매가, 카테고리, 대표 이미지, 재고 수량) 누락 시 저장을 제한한다
- 판매가/재고 수량에 음수 또는 잘못된 형식 입력 시 저장을 제한한다
- 이미지 업로드 실패 시 저장을 제한한다
- 이미 삭제된 상품 수정 시도 시 오류 페이지로 이동

### 비고

- 재고 관리 별도 화면 대신 본 화면에서 통합 관리 (개발 범위 축소)
- 카테고리 관리 별도 화면 없이 등록 시 드롭다운으로 선택
- 재고 증감 이력 로그는 이번 스프린트 제외
