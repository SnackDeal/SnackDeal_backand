# 상품 상세 페이지 (수정됨)

사용자/관리자 구분: 사용자
연관화면: 상품 페이지
도메인: user/product
기능설명: 상품 정보를 조회한다
우선순위: 상
API : /product/{product_id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/product%20%7Bproduct_id%7D%205b1bdfb849fa822c81ee81f73a09b957.md)

### 선행조건

- 유효한 상품 ID로 접근해야 한다

### 트리거

- 상품 목록에서 상품 클릭

### 동작

[상품상세정보](https://app.notion.com/p/390bdfb849fa80e997b5dd5b55dd1716?pvs=21) 

- 상품 정보 표시: 대표 이미지 1장(product_image 1행), 상품명, 가격, 상세 설명, 재고 상태
- 장바구니, 바로구매 버튼 제공
- **옵션 없음** (과자 쇼핑몰) → 수량 선택 UI만 제공
- 실시간 총 결제 예정 금액 계산 및 표시(결제창에서 확인하는걸로……)
- 하단 : Q&A , 배송 / 교환 /환불 안내(cafe24 에서 보고 복사하기)

### 예외 / 제한사항

- 존재하지 않는 상품 ID 접근 시 오류 페이지를 표시한다
- 재고 0(품절) 상품은 구매 버튼을 비활성화하고 "품절"로 표시한다(버튼)
- (선택)판매중지 상품은 접근을 차단한다
- 재고 초과 수량 선택 시 최대 재고 수량으로 제한한다
    - 최대수량 제한 x
    - 1인 구매 수량 제한 x
    - 장바구니도 재고 연동
