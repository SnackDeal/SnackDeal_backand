# 관리자_상품관리 리스트(재고 포함)

사용자/관리자 구분: 관리자
연관화면: 상품관리
도메인: admin/product
기능설명: 기존 관리자가 상품관리 리스트에 진입한다
우선순위: 상
API : admin/product (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20product%206bcbdfb849fa83b3b45001cde12e117c.md), /admin/product/{id}/status (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20product%20%7Bid%7D%20status%20034bdfb849fa831b8afa8140e4153eb6.md)

### 선행조건

- 관리자 로그인 상태여야 한다

### 트리거

- 상품관리 메뉴 진입

### 동작

[상품관리 리스트](https://app.notion.com/p/390bdfb849fa8077a22aea496e7e9295?pvs=21) 

- 등록된 상품 목록을 표시한다
    - 컬럼: 썸네일(대표이미지1개), 상품명, 카테고리, 판매가, 재고 수량, 판매 상태, 등록일
- 검색 및 필터 기능
    - 상품명 검색
    - 카테고리 필터(대/중/소)
    - 판매 상태 필터 (판매중 / 품절 / 판매중지) 체크박스
    - "재고 부족" 필터 (재고 10개 이하)
- 정렬 기능
    - 최신순, 재고 낮은 순, 재고 높은 순, 판매가 순
- 리스트 상에서 판매 상태를 바로 변경할 수 있다 (드롭다운)
- 각 행에서 상품 수정 화면으로 이동한다
- 상단에 "상품 등록" 버튼을 배치한다

### 예외 / 제한사항

- 조회 결과가 없을 경우 빈 결과를 표시한다
- 판매 상태 변경 실패 시 이전 상태로 롤백하고 안내한다
