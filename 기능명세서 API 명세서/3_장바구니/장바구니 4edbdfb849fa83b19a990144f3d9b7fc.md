# 장바구니

사용자/관리자 구분: 사용자
연관화면: 장바구니
도메인: user/cart
기능설명: 기존 사용자가 장바구니에 등록한 상품을 조회한다
우선순위: 상
API : /cart (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cart%20361bdfb849fa823795bf815160622d8a.md), /cart (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cart%20ca7bdfb849fa83c7ad0881ec3108f3b2.md), /cart/{item_id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cart%20%7Bitem_id%7D%20d23bdfb849fa82d0b8c10166c901fc19.md), /cart (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cart%20a71bdfb849fa82ea920e01953d76cf04.md)

### 선행조건

- 로그인 상태여야 한다

### 트리거

- "장바구니 담기" 클릭 / 장바구니 페이지 접속

### 동작

- 담은 상품 목록 표시: 썸네일(대표 이미지), 상품명,  수량(선택할수 있게), 개별 금액
- 하단에 상품 총 금액과 예상 결제 금액을 표시한다
- 수량 변경 및 개별 삭제 기능 제공
- 선택 삭제 및 전체 삭제 기능 제공
- "주문하기" 버튼 클릭 시 주문정보입력 페이지로 이동

### 예외 / 제한사항

- 비로그인 상태에서 "장바구니 담기" 클릭 시 로그인 페이지로 이동한다
- 재고 초과 수량 입력 시 최대 재고 수량으로 제한한다
- 담긴 상품이 품절/판매중지 상태로 바뀐 경우 결제 진행을 차단하고 안내한다
- 마감임박일경우(10개미만) 옆에 마크 or 안내 넣기, (00개 남음 글 빨간색으로 표시)

### 비고

- 할인 적용 금액은 주문서작성의 결제예정금액에서 확인 가능합니다.
- 회원 전용 기능 (서버 저장 방식)
