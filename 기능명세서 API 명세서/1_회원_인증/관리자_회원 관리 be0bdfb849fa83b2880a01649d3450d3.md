# 관리자_회원 관리

사용자/관리자 구분: 관리자
연관화면: 회원정보
도메인: admin/members
기능설명: 기존 관리자가 회원관리 페이지에 진입한다
우선순위: 중
API : /admin/members (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20members%2028abdfb849fa8226a7dd812dfe6551d5.md), /admin/members/{id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20members%20%7Bid%7D%2059cbdfb849fa83118006818657a284c0.md), /admin/members/{id}/status (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20members%20%7Bid%7D%20status%2069ebdfb849fa83089aa7015eadccc4d6.md)

### 선행조건

- 관리자 로그인 상태여야 한다

### 트리거

- 회원 관리 메뉴 진입

### 동작

[관리자 - 회원관리](https://app.notion.com/p/390bdfb849fa802c93b6c6a43d66c007?pvs=21) 

- 회원 목록을 표시한다
- 컬럼: 이메일, 이름, 가입일, 최근 접속일, 상태
- 이메일/이름으로 검색한다
- 회원 상태 필터 (정상 / 휴면 / 탈퇴)
- 회원 상세에서 상태를 변경한다 (정상 → 휴면, 탈퇴 처리)

### 예외 / 제한사항

- 관리자 본인 계정에 대한 상태 변경은 차단한다

### 비고

- 상세 프로필 수정 기능은 제외, 상태 변경 중심의 최소 스펙
- 누적 지표/최근 주문은 시간 남으면 붙이는 선택 요소
