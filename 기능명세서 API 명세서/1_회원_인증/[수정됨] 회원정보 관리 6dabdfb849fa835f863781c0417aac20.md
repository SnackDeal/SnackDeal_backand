# 회원정보 관리 (수정됨)

사용자/관리자 구분: 사용자
연관화면: 마이페이지, 상단페이지
도메인: user/member
기능설명: 기존 사용자가 회원정보를 확인 및 수정한다
우선순위: 상
API : /member/me (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/member%20me%207d3bdfb849fa8266969f81c344f13d3c.md), /member/me (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/member%20me%20879bdfb849fa83a4bca6017b8bb05f76.md)

### 선행조건

- 로그인된 상태여야한다.

### 트리거

- 마이페이지> 회원정보관리 , 상단메뉴 > 회원정보관리(아이콘)

### 동작

```jsx
[프로세서]
비밀번호 확인 -> 회원정보 -> 비밀번호변경/휴대폰번호 변경
```

- 처음에 비밀번호 확인해서 비밀번호가 맞으면 회원정보 페이지에 진입한다
- 회원 정보 페이지에서 휴대폰번호(phone), 비밀번호 변경이 가능하도록 한다
- 이름/이메일/생년월일/성별은 조회만 가능 (수정 불가)
- **배송지(주소)는 회원정보가 아닌 별도 [주소록(delivery)] 화면에서 관리**한다 (여러 배송지 저장·기본 배송지 지정)

### 예외 / 제한사항

- 비밀번호 불일치
- 필수입력사항 확인
