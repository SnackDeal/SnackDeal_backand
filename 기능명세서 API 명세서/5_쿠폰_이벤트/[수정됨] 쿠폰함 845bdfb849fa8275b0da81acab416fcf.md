# 쿠폰함 (수정됨)

사용자/관리자 구분: 사용자
연관화면: 마이페이지
도메인: user/mypage
기능설명: 기존 사용자가 가진 쿠폰함을 확인한다
우선순위: 상
API : /mypage/coupon (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/mypage%20coupon%2035abdfb849fa83d798d0017ca87fd2fd.md)

### 선행조건

- 로그인 상태여야 한다

### 트리거

- 마이페이지 > 쿠폰함 진입

### 동작

- 보유 쿠폰(user_coupon) 목록을 탭으로 구분하여 표시한다 (사용 가능=ACTIVE / 사용 완료=USED / 만료=EXPIRED)
- 각 쿠폰 항목: 쿠폰명, 할인 방식(FIXED/PERCENT), 유효기간, 사용 조건(min_order_price), (선택)발급 경로(issue_type: SIGNIN/EVENT)
- 쿠폰 등록 경로
    - 회원가입 시 자동 발급 (issue_type=SIGNIN)
    - 이벤트게시판에서 사용자가 직접 다운로드 (issue_type=EVENT)

### 예외 / 제한사항

- 유효기간 지난 쿠폰
- 발급 수량 소진
- 이미 발급된 쿠폰

### 비고

- 쿠폰 발급 경로가 다양하므로, 사용자 계정에 발급된 쿠폰은 모두 쿠폰함에서 통합 조회 가능해야 함
