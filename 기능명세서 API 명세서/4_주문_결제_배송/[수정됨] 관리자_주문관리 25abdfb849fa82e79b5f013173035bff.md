# 관리자_주문관리 (수정됨)

사용자/관리자 구분: 관리자
연관화면: 주문관리
도메인: admin/order
기능설명: 기존 관리자가 주문관리페이지에 진입한다
우선순위: 상
API : /admin/order (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20order%20e73bdfb849fa8307953981caf6dda782.md), /admin/order/{id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20order%20%7Bid%7D%20643bdfb849fa8288a2aa01383314cf6c.md), /admin/order/{id}/refund (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20order%20%7Bid%7D%20refund%20c8abdfb849fa83f1a58581fd7b316c3b.md), /admin/order/{id}/status (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20order%20%7Bid%7D%20status%20dbdbdfb849fa8314b1df0115bd4848ec.md)

### 선행조건

- 관리자 로그인 상태여야 한다

### 트리거

- 주문관리 메뉴 진입

### 동작

[관리자 - 주문관리 ](https://app.notion.com/p/390bdfb849fa80029e6dcc772b9b6cc9?pvs=21) 

- 전체 주문 목록을 최신순으로 표시한다
    - 컬럼: 주문번호, 주문일자, 회원명, 상품(대표), 결제금액, 주문 상태
- 검색 및 필터 기능
    - 주문번호 검색, 회원명 검색
    - 주문 상태 필터 (결제완료 / 배송준비 / 배송중 / 배송완료 / 취소)
    - 기간 필터
- 주문 상세에서 주문 상태를 변경한다 (예: 배송준비 → 배송중 → 배송완료)
- 주문 상세 정보 확인: 상품 목록, 수량, 배송지, 결제 내역, 사용 쿠폰 (과자몰은 옵션 없음)

### 예외 / 제한사항

- 이미 배송완료된 주문의 상태를 역방향으로 되돌리는 경우 확인 절차를 거친다
- 취소 처리 시 재고 복구 여부에 대한 정책 확정 필요

### 비고

- 환불/취소 처리는 관리자 수동 대응
