# 관리자_대시보드

사용자/관리자 구분: 관리자
연관화면: 메인페이지
도메인: admin
기능설명: 기존 관리자가 로그인 후 대시보드에 진입한다
우선순위: 상
API : /admin/main (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20main%20843bdfb849fa827dac7601496b3bee2d.md)

### 선행조건

- 관리자 로그인 상태여야 한다

### 트리거

- 관리자 페이지 접속

### 동작

- 핵심 지표를 요약 카드로 표시한다
    - 금일 주문 수
    - 금일 매출
    - 신규 회원 수(오늘 / 전체)
    - 재고 부족 상품 수 (재고 10개 이하) (선택)
    - 미처리 문의 수
- 각 카드 클릭 시 해당 관리 화면으로 이동한다

### 예외 / 제한사항

- 데이터가 없을 경우 0으로 표시한다

### 비고

- chartjs(그래프)
- 상세 통계 화면은 이번 스프린트 제외
