# 관리자_로그인

사용자/관리자 구분: 관리자
연관화면: 로그인
도메인: admin
기능설명: 관리자가 로그인 페이지에 진입한다
우선순위: 상
API : /admin/login (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/admin%20login%2085dbdfb849fa82369681019d2a02052f.md)

### 선행조건

- 등록된 관리자 계정이어야 한다

### 트리거

- 관리자 로그인 폼 제출

### 동작

- 입력값 유효성 검증
- 인증 성공 시 관리자 세션을 생성한다
- 인증 실패 시 오류 메시지를 표시한다

### 예외 / 제한사항

- 이메일 또는 비밀번호 불일치
- 권한이 없는 계정의 접근을 차단한다

### 비고

일반 사용자 계정 체계와 분리 권장(Role)
