# 공지사항게시판 (수정됨)

사용자/관리자 구분: 사용자
연관화면: 고객센터
도메인: user/cs
기능설명: 사용자가 공지사항 게시판을 확인한다
우선순위: 상
API : /cs/notice/list (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cs%20notice%20list%20f32bdfb849fa8209974201617bab5a7b.md), /cs/notice/{id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cs%20notice%20%7Bid%7D%205b8bdfb849fa83ee9f7881fabf3b68b3.md)

### 선행조건

- 회원 x

### 트리거

- 고객센터 > 공지사항게시판

### 동작

- 일반 게시판형태 (목록 + 상세, is_pinned 상단 고정)
- 노출여부 설정 추가 (deleted_at으로 소프트 삭제)

> notice 테이블(id, title, content, is_pinned, created_at/updated_at/deleted_at) 기준

### 예외 / 제한사항

- 게시판형태와 동일
