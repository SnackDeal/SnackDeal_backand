# 고객센터 챗봇페이지

사용자/관리자 구분: 사용자
연관화면: 고객센터
도메인: chatbot(도메인)
기능설명: 사용자가 고객센터 챗봇페이지에서 질문을 한다
우선순위: 상
API : /chatbot/ask (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/chatbot%20ask%20235bdfb849fa831b8afe815d51c24641.md)

### 선행조건

- 회원 x

### 트리거

- 페이지 하단에 아이콘(챗봇)

### 동작

- 단순 QNA만 하도록함(ai쪽에서 api작업)

### 예외 / 제한사항

- ai 연결 오류
- (선택) 3개정도 질문중에 선택
