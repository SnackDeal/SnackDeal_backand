# 문의하기 (수정됨)

사용자/관리자 구분: 사용자
연관화면: 고객센터, 마이페이지
도메인: user/cs
기능설명: 기존 사용자가 문의하기 게시판을 이용한다
우선순위: 중
API : /cs/qna (POST, 문의 등록) (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cs%20qna%20create.md), /cs/qna/faq (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cs%20qna%20faq%20e8abdfb849fa833aa008816c6b258e5d.md), /cs/qna/{id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cs%20qna%20%7Bid%7D%204e3bdfb849fa82ddb0b801d7b5b83d30.md),  /cs/qna/{id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cs%20qna%20%7Bid%7D%20c90bdfb849fa838a9c11018399701dfa.md), /cs/qna/{id} (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cs%20qna%20%7Bid%7D%20e2dbdfb849fa82cfb7e08164b50c5c92.md), /cs/qna/list (../API%20%EC%A0%95%EC%9D%98%EC%84%9C/cs%20qna%20list%201eabdfb849fa835da0d00144ef844d1c.md)

### 선행조건

- 로그인 상태여야 한다

### 트리거

- 고객센터 > 문의하기 진입

### 동작

- 기본리스트,  상세페이지 + 답변(관리자만 가능)

---

```jsx
                    [문의하기 버튼]
                    
   FAQ 목록
   전체 회원 주문.......
```

- FAQ로 해결되지 않을 경우 문의 폼을 통해 접수한다
    - 입력 항목: 문의 유형(type: `ORDER` 주문 / `SHIPPING` 배송 / `PRODUCT` 상품 / `OTHER` 기타), 제목(title), 내용(content), 첨부(attachment_url, 선택)
    - 문의 등록: **POST /cs/qna** (Body: type, title, content, attachment_url)
- 문의 접수 완료 시 안내 메시지를 표시한다
- 본인 문의 내역과 답변 상태를 조회할 수 있다

### 예외 / 제한사항

- 비로그인 상태에서 문의 폼 접근 시 로그인 페이지로 이동한다
- 필수 입력값(유형, 제목, 내용) 누락 시 제출을 제한한다
- 답변이 달린경우 삭제/수정 불가

### 비고

- 회원 전용 기능
